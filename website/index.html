Frequently Asked Questions

Does MobileCLI require root?
No! MobileCLI works without root on any Android phone. It uses Android's permission system to access hardware features, giving you near-complete control without modifying your system.
Can I actually build Android apps on my phone?
Yes! MobileCLI includes Java 17, Gradle, and the Android SDK. You can write, compile, and install APKs directly on your device. The app can even rebuild itself.
Which AI assistants are included?
Pro users get access to Claude (Anthropic), Gemini (Google), Codex (OpenAI), and OpenRouter (100+ models). Each has its strengths ‚Äî Claude for reasoning, Gemini for context, Codex for generation.
What phones are supported?
MobileCLI works on any Android phone running Android 7.0 (Nougat) or newer. We recommend Android 10+ for the best experience. Both ARM and ARM64 devices are supported.
Is it safe to give an app 79 permissions?
MobileCLI only uses permissions when you explicitly call a command. The app doesn't run background services collecting data. All operations are user-initiated through the terminal.
Can I use MobileCLI offline?
Yes! The terminal, package manager, and all hardware commands work offline. AI assistants and deployment CLIs require internet, but core functionality is fully offline-capable.
